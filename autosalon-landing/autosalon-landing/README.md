# Auto Salon Landing Page (for verification)

Use this public page as the **Your website** link in Lemon Squeezy verification.

## Deploy on GitHub Pages (easy)

1) Create a new GitHub repository (Public)
   - Example name: `auto-salon-script`

2) Upload these files to the repo root:
- `index.html`
- `style.css`
- folder `assets/`

3) Enable GitHub Pages:
- Repo **Settings** → **Pages**
- Source: **Deploy from a branch**
- Branch: **main** / folder: **/(root)**
- Save

4) GitHub will show you a URL like:
`https://YOURUSERNAME.github.io/auto-salon-script/`

5) Put that URL into Lemon Squeezy → **Business details** → **Your website**.
